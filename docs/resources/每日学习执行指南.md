# 🎯 每日学习执行指南（面试满分版）

> **目标**：后端 + AI Agent 双栈面试满分。
> **原则**：每天照做就行，不用自己决定学什么。完成后打勾即可。
> **节奏**：工作日每天 2 小时（学30min + 验10min + 练70min + 记10min），周末 3-4 小时。
> **更新方式**：每天开始学习前打开本文件，按当天任务执行；学完回来打勾；遇到问题随时问 AI 助教。

---

## 📌 每日标准四步流程（所有天都要做）

```
□ Step1 学（20-30min）：看指定资料，合上书能用自己话讲清楚
□ Step2 验（5-10min）：运行抽题命令，口述答题，对照速查打★错题
□ Step3 练（60-120min）：AI 实战，跑代码/写项目，遇到问题问助教
□ Step4 记（5-10min）：回来勾掉完成项，写一句话收获/卡点
```

## 🔥 WEEK 1 · 启动周（7/3 周五 ~ 7/5 周日，3天）

**本周目标**：跑通 AI 环境 + 手写 RAG + 后端热身 8 题。里程碑：「能让 AI 听我指挥，我知道怎么学了」。

### 📅 Day 1 · 周五 7/3（今天，约 1.5h）

**Step1 学（20min）**
- [ ] 快速翻看 [后端面试满分速查 §MySQL](后端面试满分速查.md) Q1-Q3（B+Tree/聚簇索引/MVCC），温故知新，不要死记
- [ ] 了解一下什么是 Prompt、Token、Temperature 三个基本概念（问助教"1分钟解释Token和Temperature"即可）

**Step2 验（5min）**
- [ ] 终端跑 `./scripts/quiz.sh mysql 2`，抽 2 道 MySQL 题，**口述**答一遍
- [ ] 答完对照速查手册，卡壳的题标★

**Step3 练（60min）—— AI 实战 Hello World**
- [ ] 进入 `projects/week1-quickstart/`
- [ ] 装依赖：`pip install openai python-dotenv rich`（用 uv 或 pip 都可以）
- [ ] `cp .env.example .env`，编辑 `.env` 填入 API Key 和 Base URL（没有 Key 问助教推荐国内便宜方案）
- [ ] 运行 `python 01_hello_ai.py`，看到 AI 在终端回你话
- [ ] 试着改一下 system prompt 让它用不同语气回答（比如"用古诗回答"）

**Step4 记（5min）**
- [ ] 回这个文件打勾完成项
- [ ] 写一句话：今天最大收获/卡在哪？
- [ ] 告诉助教 "Day1 完成了，我的感受是___"

### 📅 Day 2 · 周六 7/4（约 2.5h）

**Step1 学（25min）**
- [ ] 后端：看 [后端面试满分速查 §Redis](后端面试满分速查.md) Q1-Q3（为什么快/缓存三问题/分布式锁）
- [ ] AI：理解 CoT 和 Function Calling 概念——看 `02_prompt_playground.py` 和 `03_function_calling.py` 源码开头注释，重点理解「LLM 不执行工具，它只生成调工具的 JSON，真正执行是你 Python 代码」

**Step2 验（10min）**
- [ ] `./scripts/quiz.sh redis 3`，抽 3 道 Redis 题口述
- [ ] `./scripts/quiz.sh llm基础 1`，抽 1 道 LLM 基础题

**Step3 练（100min）**
- [ ] 跑 `python 02_prompt_playground.py`，观察三种 Prompt 的效果差异。**关键：把里面的测试问题换成你自己的问题（比如一道业务场景题），对比 Zero-shot 和 CoT 结果**
- [ ] 跑 `python 03_function_calling.py`，读代码理解 ReAct 循环（while 循环里 LLM 决策→调工具→结果喂回→再决策）
- [ ] **动手加分**：在 `03_function_calling.py` 里加一个你自己的工具（比如 `get_current_time()` 返回当前时间，或 `query_db(sql)` 假数据查询），让模型学会调用它
- [ ] 跑 `python 04_streaming_chat.py`，和你自己的 AI 聊几句，观察流式打字机效果

**Step4 记（10min）**
- [ ] 打勾 + 写一句话收获
- [ ] 重点思考：Agent 本质就是个循环吗？循环怎么终止的？把你理解写下来

**Step1 学（25min）**

- [ ] 挑战：试着问一个 DOCUMENTS 里**没有**答案的问题，看 AI 会怎么回答？会幻觉吗？怎么解决？（这个问题 W2 会深入讲）

- [ ] `git add -A && git commit -m "W1完成: HelloAI→Prompt→FC→流式→手写RAG"`
