# 📊 后端+AI 能力自检矩阵

> 每月初用这份矩阵给自己打分（1-5 分），低于 3 分的优先补。这是你的"能力仪表盘"。
>
> 打分标准：
> - **1 分**：听说过，不了解
> - **2 分**：了解概念，没做过
> - **3 分**：做过 Demo / 跟教程做过
> - **4 分**：独立做过项目，能讲清原理和踩坑
> - **5 分**：生产落地过，能指导他人，能做选型决策
